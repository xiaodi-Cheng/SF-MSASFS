
The source code is distributed for academic purposes only.

This page contains the code of the SF-MSASFS algorithm presented in the following paper along with the corresponding experimental results:

“Multi-Surrogate-Assisted Stochastic Fractal Search Based on Scale-free Network for High-Dimensional Expensive Optimization”

The 'main.m' file is the main execution file. Please note that it is necessary to set the file paths before running the code. 
Use the following commands to set the paths:
addpath('E:\test function matlab files')%Set Corresponding Path
Replace  'E:\test function matlab files' with the corresponding file path for  the folder containing MATLAB files related to the test functions.
